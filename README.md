# IB E-Source

An interactive React + Vite web app that curates visual arts, theatre, music, and dance references for IB students. Browse, search, and open analysis guides with direct links to authoritative sources.

## Getting started

```bash
npm install
npm run dev
```

Visit the printed local URL to explore the archive. Use the subject chips to filter, the search bar to find artists, works, or concepts, and open an item to read the contextual, formal, and significance notes. The About modal highlights the creator's journey.

## Building

```bash
npm run build
```

Outputs a production-ready bundle in `dist/`.
